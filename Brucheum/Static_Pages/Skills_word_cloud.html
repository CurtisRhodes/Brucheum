I went to sleep last night with an idea, I work this morning and built this page. It's just a display form. The data comes from a list that comes from a table. Once you've created one data entry form and a standard GET, api call that retrieves the data, all that needs to be done is copy and change the url, and table and field names. This time instead of just populating a list or a drop down I wanted a different look for my display.
It's all about how it looks. As a contractor i learned long ago that you can write an app that lands a rocket on the moon but if the interface doesn't look good, I'm sorry, but your customers are just not going to fall in love with your app than they love one with a flashy front end even if it doesn't actually do anything. 
For this form I let CSS do all the work.  Inside the loadData() function it appends a bunch of keywords in <div> tags to a container div. The word <div> position is set to absolute. Outside the append loop a var posTop is set to the position().top of the container. Another var named posLeft is set to 0. Inside the loop each item has its "top" and "left" style attributes set to these variables. After each tag is appended its width is added to posLeft.  If posLeft exceeds the right margin it is reset and the posTop is incrimented by the value of a new line. 
  That's it. Hey I said I wrote this one day. I tried using the modulus operator to take every seventh or so word and display it vertical as they do in some word clouds, but I had trouble with line spacing on the following rows. So we're done. But there is one more thing to mention which is the most distinctive and perhaps the most visually satisfying feature, the variable font size of the word elements. I had a row in my Skill table called SortOrder because of an earlier design idea. I was going to determine the font size based on the value of a Proficiency field which is a lookup field that would have to be translated. instead I use the number I put in the sortOrder field as the actual font size. Anywhere from 10 to 22 or whatever. In the loop where I set the top and left styles of the <div> tag I also set the font-size style to this value.   
  I also added a click event to the word elements that displays a dialog box with narrative details about my particular experience with each skill. Now if I just stop fiddling with this website and start using it maybe I'll fill in some of that dialog.
